# Registro de Usuarios con PHP y MySQL

Este es un proyecto básico de registro de usuarios desarrollado con **PHP** y **MySQL**, utilizando **XAMPP** como entorno local de desarrollo.
Características

- Formulario de registro con validación de campos.
- Sanitización de datos de entrada con `filter_input`.
- Hash de contraseñas con `password_hash()` (algoritmo `bcrypt`).
- Inserción segura con **prepared statements** usando `mysqli_prepare()`.
- Separación de la lógica de conexión (`database.php`) y la lógica de presentación (`index.php`).
 Tecnologías

- PHP 7+
- MySQL / MariaDB
- HTML
- XAMPP (Apache + MySQL)
